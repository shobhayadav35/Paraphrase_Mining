Paraphrase Mining Using Sentence Transformers

This project demonstrates how to identify semantically similar sentences using Sentence Transformers and Natural Language Processing (NLP) techniques. It leverages pre-trained transformer models to generate sentence embeddings and performs paraphrase mining to discover sentence pairs with similar meanings.

The notebook covers the complete workflow of semantic text analysis, including sentence embedding generation, cosine similarity computation, paraphrase detection, multilingual embedding support, and outlier sentence identification. It uses lightweight yet powerful transformer models such as all-MiniLM-L6-v2 and distiluse-base-multilingual-cased-v1 for efficient semantic understanding across different languages.

Features
Generate high-quality sentence embeddings
Perform paraphrase mining on multiple sentences
Measure semantic similarity using cosine similarity
Detect semantically similar sentence pairs
Support multilingual sentence embeddings
Identify outlier sentences based on similarity scores
Easy-to-understand Jupyter Notebook implementation
